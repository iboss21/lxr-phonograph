When adding new screenshots:

1. **Follow naming convention**: `[category]-[subject]-[variant].png`
2. **Use correct subdirectory**: Place in appropriate folder
3. **Create metadata file**: Add `.txt` file with description
4. **Update documentation**: Reference in relevant .md files
5. **Verify quality**: Check resolution, clarity, and file size
